# Text Editor Project

This project involves building a basic text editor in Java using Java Swing components. The text editor allows users to create, open, edit, and save text files.

## Features

- **File Menu:**
  - New File: Create a new empty text file.
  - Open File: Open an existing text file.
  - Save File: Save the current text to a file.

- **Edit Menu:**
  - Cut: Cut selected text.
  - Copy: Copy selected text.
  - Paste: Paste copied or cut text.
  - Select All: Select all text in the editor.
  - Close: Close the text editor application.

## Project Structure

- **TextEditor.java:** Main class containing the text editor application.
- **TextEditorUI.java:** Class responsible for building the UI of the text editor.
- **ActionListenerImpl.java:** Implementation of ActionListener interface for handling menu actions.
- **FileMenuHandler.java:** Implementation of file menu functionalities.
- **EditMenuHandler.java:** Implementation of edit menu functionalities.

## Usage

1. Clone the repository to your local machine.
2. Open the project in IntelliJ IDEA.
3. Run the `TextEditor.java` file to launch the text editor application.
4. Use the file and edit menus to perform various actions such as creating, opening, editing, and saving text files.

## Dependencies

- Java Development Kit (JDK) 8 or later
- IntelliJ IDEA or any Java IDE supporting Swing
